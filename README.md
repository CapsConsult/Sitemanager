# Sitemanager

Prototype for placing pins with photos on ship PDF drawings, built for shipyard site management.

## MVP goal

Build a mobile-friendly web app where users can:

- open a PDF drawing
- place pins on the drawing
- attach photos to pins
- view saved pins on phone and laptop

## First scope

For the first version, only these features are needed:

- PDF upload/view
- place pins on a PDF
- attach one or more photos to a pin
- save data locally in the browser

## Later phases

Possible next phases:

- shared cloud storage
- tasks and status per pin
- planning and activity requests
- clash detection
- user roles and permissions
